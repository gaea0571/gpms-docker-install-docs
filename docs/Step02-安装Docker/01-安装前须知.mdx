

## 系统要求

- Windows 11 64 位：专业版 21H2 或更高版本，或者企业版或教育版 21H2 或更高版本。
- Windows 10 64 位：Pro 21H1（内部版本 19043）或更高版本，或者 Enterprise 或 Education 20H2（内部版本 19042）或更高版本。
